## 🚀 Starting

Be welcome! In this project, a complete CRUD in Mern Stack

## 🎬 Methods

As a methodology I sought a simple syntax so that readers can easily understand the fundamentals of CRUD in Mern.

* Node Js
* Express
* MongoDB
* React
* Axios
